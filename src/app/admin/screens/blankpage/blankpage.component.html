Key Processes in this Application

Find out which module you want to work - in SRC folder

If you want to work in STORE Module then click on app/store folder

Find out screens folder -> this is parent component

1.If you want to create new component then type this command in terminal----
ng g c store/screens/ new_component_name
Ex: ng g c store/screens/test

2.After created new component then click on Menu folder which is existing in store module,open menu.component.html page and add menu item list

Ex:
<li class="nav-item">
<a class="nav-link collapsed" href="store/testmaster">
<i class="bi bi-dash-circle"></i>
<span>test Master</span>
</a> </li>

3.And finally we creating routing now
click on store-routing.module.ts which in the same store module,open this file and add this code,

path:'itemmaster',
component: ItemmasterComponent
